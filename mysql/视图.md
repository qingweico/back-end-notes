## 视图

### 视图的创建

```mysql
create VIEW my_view
AS 
select name,salary from job;
```

视图的好处：

- 重用sql语句
- 简化复杂的sql操作，不必知道内部的细节
- 保护了数据，提高的安全性

### 视图的修改

方式一：

```mysql
create or replace view 视图名 as 查询语句;
```

方式二：

```mysql
alter view 视图名 as 查询语句;
```

### 视图的删除

```mysql
drop view 视图名,视图名,....;
```

### 查看视图的结构

```mysql
desc 视图名；
show create view 视图名;
```
